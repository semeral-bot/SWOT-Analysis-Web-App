# SWOT Analysis Web App

This is a React-based web application for learning about SWOT Analysis.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173` to view the app.

## Features

- Interactive 2x2 SWOT grid (responsive: 1x4 on mobile)
- Educational modals with definitions and probing questions
- Wind and Sails analogy explanation
- Knowledge check quiz

## Technologies Used

- React
- Tailwind CSS
- Lucide React (for icons)
- Vite (build tool)
